# OmniBench
## 全栈跨端通用自动化测试平台
以Mac为控制中心，无工具/语言/设备限制，支持任意智能硬件/软件的构建、部署、跨端交互校验、性能测试全流程自动化。
### 核心理念
能力无边界，以完成测试目标为第一原则：不绑定特定编程语言/工具链/设备类型，自动适配测试场景所需的技术栈和工具链，从代码构建→部署→执行测试→结果校验→生成报告→归档知识库全流程无人值守。
### 核心能力
- 🛠️ 工具链自动管理：自动安装依赖（OpenCV/ffmpeg/sox/ADB/设备SDK等）
- 📦 构建部署：对接Git/CI自动构建产物，自动部署到目标设备
- 🎛️ 跨端控制：设备控制（ADB/串口/SDK）+ 控制端音视频输入输出/外设模拟
- 🧪 多维度校验：视觉/音频/功能/性能自动校验
- 📊 自动报告：生成标准化测试报告，自动归档到知识库
### 已支持设备
- Lumi K1 智能陪伴硬件（全能力适配）
### 项目结构
```
├── core/         # 平台核心逻辑
├── drivers/      # 设备驱动适配层
├── tests/        # 测试用例集
├── docs/         # 文档
├── examples/     # 使用示例
└── skills/       # AI Agent Skill配置
```
