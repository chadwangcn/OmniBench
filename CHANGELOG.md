# Changelog
## v0.2.0 (2026-07-08)
### 新增功能
- ✨ 版本管理和自动升级能力，支持`upgrade`命令一键更新
- ✨ Figma对接支持MCP优先，自动回退到API Token方式
- ✨ UI Automator能力集成，支持UI层级导出、按文本点击
- ✨ 截图自动适配定制ROM，兼容K1等定制Android系统
- ✨ 自动环境检测和依赖安装
- ✨ 原子化测试工具集：安装/日志/截图/录屏/点击/滑动/遍历/稳定性测试/设计稿对比
- ✨ 测试结果自动归档到Obsidian知识库
- ✨ 支持配置文件统一管理API Key和全局参数
### 平台特性
- 定位为纯原子工具集，所有决策逻辑由调用Agent完成
- 无内置模型/推理逻辑，轻量通用
- 支持扩展任意设备/测试场景
